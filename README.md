# 项目简介
微信小程序的广科大社区团购系统由用户微信端、服务端两大核心模块构成，功能覆盖用户、管理员、团长三类角色。用户微信端提供首页浏览、商品信息查看、购物车操作及“我的”功能，可实现收藏、自提点选择、团长评级、客服聊天、团长评价、充值、查看聊天记录与修改密码等操作。服务端对接管理员与团长权限，管理员拥有首页、客服聊天、团长评级、商品信息管理、订单管理、轮播图配置、自提点管理及用户信息管理等全流程管控功能；团长则可通过首页操作，进行订单管理、团长评级及商品信息管理，三方协同实现社区团购的用户消费、团长运营与平台管理一体化，满足社区团购的全场景业务需求。


## 核心功能
- 用户微信端
  - 首页浏览
  - 商品信息查看
  - 购物车操作
  - “我的”功能（收藏、自提点选择、团长评级、客服聊天、团长评价、充值、查看聊天记录、修改密码）
- 服务端
  - 管理员权限
    - 首页
    - 客服聊天
    - 团长评级
    - 商品信息管理
    - 订单管理
    - 轮播图配置
    - 自提点管理
    - 用户信息管理
  - 团长权限
    - 首页操作
    - 订单管理
    - 团长评级
    - 商品信息管理

## 适用场景
- 社区团购业务
- 用户消费场景
- 团长运营场景
- 平台管理场景

## 技术特点
- 微信小程序平台
- 用户、管理员、团长三类角色权限管理
- 用户消费、团长运营与平台管理一体化
- 全场景业务需求满足
